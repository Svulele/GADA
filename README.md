**GADA**

GADA is a lightweight asset tracking and chain-of-custody system designed to log, trace, and audit the movement of physical assets in real time.

It provides visibility into who handled an asset, where it was moved, and when the action occurred.

**Features**

-Asset registration via unique tag
-Scan and log asset movements
-Timestamped audit trail (immutable event log)
-Last known location tracking
-User-based scan logging
-Dashboard with search functionality
-Full movement history per asset

**Tech Stack**

Node.js

Express.js

SQLite (better-sqlite3)

Vanilla HTML, CSS, JavaScript

Nodemon (development)
